#收录近乎所有中国大学名称（一共2901所）

返回的皆为json数据

**接口调用方法**

如 查询 江西省南昌市的所有大学名称

```
https://api.hcfpz.cn/un/schools?province=江西省&city=南昌市
```

如 查询 湖北省所有大学

```
https://api.hcfpz.cn/un/schools?province=湖北省
```

如 查询 所有大学列表

```
https://api.hcfpz.cn/un/schools

```

**获取拥有大学的省份**

```
https://api.hcfpz.cn/un/provinces

```

**获取某省份拥有大学的城市**

```
https://api.hcfpz.cn/un/citys?province=湖北省

```
